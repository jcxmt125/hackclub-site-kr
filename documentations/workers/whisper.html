Deploy instructions

First, create a cloudflare account.
Go to the workers/pages menu on your dashboard.

Click the blue "create" button.

Find and click the Whisper worker template. This step is required to generate Workers AI bindings.

Name your worker, and create it. (You may be asked to set a *.workers.dev domain!)

Click the edit code button immediately, or go and see your worker. From here, click the edit code button.

Paste in the code to the edit space, and click deploy. It should be ready to use!
Visit your worker to test it out.
For easier usage, go to settings, then triggers, and configure custom URLs or paths.
For wrangler (CLI) deploy, see cloudflare's documentation.

Usage instructions

This worker is an API.
Try sending a GET request to the endpoint, with the following headers:

  • fileUrl: URL of the file you'd like to transcribe
  • authKey: The preset authentication key
You should recieve a JSON response.
There is an accompanying test script.
Note: Discord CDN blocks requests from the cloudflare workers platform.
