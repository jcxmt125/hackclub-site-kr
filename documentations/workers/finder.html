배포 방법

우선, 클라우드플레어 계정을 준비해주세요.
대시보드의 workers/pages 메뉴로 향하세요.

파란색 만들기 ("create") 버튼을 누르세요.

워커 생성하기 ("Create Worker") 버튼을 눌러주세요!

워커의 이름을 짓고, 생성하세요. (*.workers.dev 도메인을 만들어 달라는 요청을 받을 수 있어요!)

바로 코드 편집 (edit code) 버튼을 누르거나, 워커의 정보를 확인하세요. 여기서, 코드 편집 버튼을 누르세요.

코드를 편집기에 붙여넣으세요.

배포 완료를 위해 조금의 추가 설정이 필요합니다.
redirectUrls 리스트를 파일 저장소의 URL로 채워넣으세요.
commonExts 를 주로 사용하시는 파일 확장자로 채워넣으세요.
워커를 배포하고, URL을 방문해 테스트해보세요. (파일을 찾는데에 약간의 시간이 걸릴거에요. 조금만 기다려주세요!)
더 쉬운 사용을 위해 설정, 트리거 (triggers), 로 향해 커스텀 URL이나 경로 (paths)를 설정하세요.
Wrangler (CLI) 배포는, 클라우드플레어의 도움말을 참고하세요.
