Deploy instructions

First, create a cloudflare account.
Go to the workers/pages menu on your dashboard.

Click the blue "create" button.

Go ahead and click the "Create Worker" button!

Name your worker, and create it. (You may be asked to set a *.workers.dev domain!)

Click the edit code button immediately, or go and see your worker. From here, click the edit code button.

Paste in the code to the edit space.

This requires more configuration on your side.
Populate the urls list with your desired URLs.
You may copy and paste the appropriate section of code to add more paths.
Deploy your worker, then visit its URL to test it out.
For easier usage, go to settings, then triggers, and configure custom URLs or paths.
For wrangler (CLI) deploy, see cloudflare's documentation.

Usage instructions

This worker is meant to be accessed from a browser.
Try visiting its URL.
You should be 302 redirected.
There is no test script for this worker.
