Environment configuration

This test script requires the following modules:

  • python-dotenv
  • requests

This test script requires a .env file:

  • WHISPER_WORKER_URL: the url your worker (API) is deployed to.
  • WHISPER_WORKER_KEY: the "authentication key" you've deployed.

Usage instructions

In a terminal: execute the Python file.
It'll print the JSON your API returns.
It is expected to be quite complicated by default.
It should have a section where your audio is transcribed.
