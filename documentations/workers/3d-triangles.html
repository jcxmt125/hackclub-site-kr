Deploy instructions

This worker requires a wrangler (CLI) deploy.
This worker uses features in beta.
Follow along cloudflare's documentation to deploy a worker that uses entry.py to serve content.
Replace entry.py with file provided.
Place create.py into the same directory.
Run npx wrangler@latest deploy as instructed on the cloudflare docs.
Visit your worker to test it out.
To deploy as if it were a webpage, like it is deployed on this website, use a path with open fallback.
Use the Python script to make a similar static HTML and deploy it to the same path to allow for static fallback.

This script can be configured more on your side.
You may change the various parameters in create.py to make the webpage look different.
