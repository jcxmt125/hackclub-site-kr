소스 코드

HTML을 보내는 워커 스크립트
HTML을 생성하는 파이썬 스크립트

배포 방법

이 워커는 Wrangler (CLI) 배포가 강제됩니다.
이 워커는 아직 베타 상태인 기능을 사용합니다.
클라우드플레어의 설명 에 따라 entry.py를 이용하응 워커를 배포하세요.
entry.py의 코드를 위의 소스 코드로 변경하세요.
create.py를 같은 디렉토레에 배치하세요.
클라우드플레어의 설명에 적혀있듯이, npx wrangler@latest deploy를 실행하세요.
워커를 방문해 테스트해 보세요.
이 사이트에 연결된 것과 같이 배포하려면, Fallback 모드가 열림으로 설정된 Path를 이용하세요.
Python 파일을 이용해 비슷한 HTML 파일을 만들고 스태틱 사이트로 해당 Path로 업로드하세요.

이 스크립트는 더 설정이 가능합니다.
create.py의 여러 설정을 변경해 웹페이지의 모양을 바꿀 수 있습니다.

사용 방법

이 워커는 웹페이지처럼 작동합니다.
URL에 방문해보세요.
새로고침하면 새로운 가능성이 표시됩니다!
이 워커에 대해서는 테스트 스크립트가 없습니다.
