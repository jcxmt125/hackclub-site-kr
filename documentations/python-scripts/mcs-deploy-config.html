빠른 셋업

"빠른" 셋업 기능을 추가했습니다!

바닐라 호환, 최적화 모드 설치 및 편의성 클라이언트 모드 일부 호환 1.21 마인크래프트 서버를 빠르게 설치합니다.
현재 디렉토리로 서버가 자동 설치됩니다.
설정을 추가하거나 변경하려면 아래를 참고해주세요.
그렇지 않다면, 이 스크립트를 사용하면 됩니다!

파이썬은 설치되어 있어야 합니다. 기본적으로 설치되어 있겠지만, 그렇지 않은 경우 먼저 설치해주세요.
오라클 aarch 서버 (무료 티어 전체 할당량의 절반 / 12gb 램 전용)의 경우 다음 명령을 실행하세요: curl -fsSL https://hackclub.jclink.link/links/mcs-tl-u | bash
우분투 x86 서버의 경우 다음을 실행하세요: curl -fsSL https://hackclub.jclink.link/links/mcs-tl-o-a | bash

서버를 시작하려면, screen이나 tmux를 열고 source start.sh 를 실행하세요.
서버 제공자의 네트워크 설정에서 포트 25565를 허용하는 것을 잊지 말아주세요.
자동으로 Minecraft EULA에 동의하게 됩니다. 꼭 읽어봐주세요.

설정 파일 준비하기

이 프로젝트는 서버에 마인크래프트 서버 설치를 돕습니다.
우선 설정 파일이 필요합니다.
전체 설정 생성 스크립트 (mcs-make-config)를 컴퓨터에서 실행해주세요.
설명에 따라 .json 파일을 만들 수 있습니다.

서버 배포

필요한 파일을 서버로 다운로드 할 수 있습니다.
마인크래프트 서버를 설치할 디렉토리로 cd합니다.
다음 명령을 실행해 서버 셋업 도우미 스크립트를 다운로드하세요:
curl -OJ https://raw.githubusercontent.com/jcxmt125/side-projects/main/python%20scripts/kr/mcs-deploy-config/mcs-deploy-config.py
다음, scp와 같은 방식으로 같은 디렉토리에 설정 파일을 업로드하세요.
자바/JRE 또한 필요합니다.

파이썬 스크립트를 서버에서 실행하세요. 추가 모듈은 필요하지 않습니다.
설명을 따라 서버에 마인크래프트 서버를 설치합니다.
화면에 출력된 커맨드를 기억하거나, source start.sh로 서버를 시작할 수 있습니다.
이제 설치 파이썬 스크립트는 삭제해도 됩니다.

오라클 서버에 배포하기

오라클에 서버를 처음 배포하는 경우, 여기를 참고해주세요.
위 가이드를 비슷하게 따르되, SSH 연결 후 위의 스크립트를 다운로드 해주세요.

마인크래프트 서버 준비 및 배포

메인 스크립트를 다운로드 한 뒤, 다음 명령을 실행합니다:
curl -OJ https://raw.githubusercontent.com/jcxmt125/side-projects/main/python%20scripts/kr/mcs-deploy-config/mcs-oraclefree-wrapper.py
설정 파일 또한 준비해주세요.

방금 다운로드한 "wrapper" 파이썬 스크립트를 실행합니다.
기다리면 나오는 내용 중 가장 최신 버젼의 jdk의 이름을 복사합니다.
제 경우, "jdk-22-headless.aarch64"였습니다.
붙여넣고, 요청시 "y"와 엔터를 눌러 yum이 Java를 설치하도록 허용합니다.
설정 파일의 이름을 타이핑한 뒤, 엔터를 누르세요.
잠시 후, 요청받으면 엔터를 눌러 nano를 열고, eula=true로 변경한 뒤, ctrl+x, y, 엔터 순으로 눌러 저장합니다.
조금만 더 기다리면 서버를 실행할 준비를 완료합니다.

데스크탑으로 돌아와주세요.
SSH 세션은 유지해주세요. 마인크래프트 서버를 아직 시작하지 않았습니다!

네트워크 셋업

이제 서버 연결을 설정합니다.
실행되고 있는 서버의 설정 화면에서, 서브넷 링크를 클릭합니다.
subnet-[date]-[time]과 같은 형식이 기본값입니다.
"Default Security list for..." 을 클릭합니다.
"Add ingress rules" 버튼을 클릭합니다.
Source CIDR 는 0.0.0.0/0, Destination Port range 는 25565로 설정합니다.
"Add ingress rules"를 눌러 저장합니다.

Ubuntu 서버에 배포하기

메인 스크립트를 다운로드 한 뒤, 다음 명령을 실행합니다:
curl -OJ https://raw.githubusercontent.com/jcxmt125/side-projects/main/python%20scripts/kr/mcs-deploy-config/mcs-ubuntu-wrapper.py
설정 파일 또한 준비해주세요.

방금 다운로드한 "wrapper" 파이썬 스크립트를 실행합니다.
apt update를 실행하고, apt upgrade를 할지 물어봅니다.
그 다음, 사용 가능한 jre를 검색합니다.
가장 최신 버젼의 headless openjdk, 제 경우 "openjdk-21-jre-headless"를 복사합니다.
붙여넣은 뒤, 요청시 "y"를 입력하고 엔터를 눌러 apt가 Java를 설치하도록 허용합니다.
설정 파일의 이름을 타이핑한 뒤, 엔터를 누르세요.
잠시 후, 요청받으면 엔터를 눌러 nano를 열고, eula=true로 변경한 뒤, ctrl+x, y, 엔터 순으로 눌러 저장합니다.
조금만 더 기다리면 서버를 실행할 준비를 완료합니다.

완료하고 연결하기

선택적으로 더 쉬운 연결을 위해 IP 주소를 도메인에 연결하세요.
Public IP address에 대한 A 레코드를 만들어주세요.
Cloudflare를 사용중이라면 이 레코드에 대해 프록시 기능 (주황 구름)을 꺼주세요.

Start and connect to the server

스크립트가 제공한 명령어를 실행합니다.
SSH 환경을 접속 해제했을 때 서버가 계속해서 실행되도록 Screen이나 Tmux에서 실행하는 것을 추천합니다.
서버가 곧 시작됩니다!
마인크래프트 클라이언트에서, 서버를 추가해주세요.
멀티플레이를 클릭하고, "서버 추가"를 클릭해주세요.
IP 주소나, 설정된 도메인을 입력합니다.
이제 연결할 수 있습니다!

서버 유지관리

모드 변경 및 업데이트하기

컴퓨터로 업데이트 설정 생성 스크립트를 다운로드합니다.
실행한 뒤, 서버에 추가하고 싶은 모드를 전부 설정합니다.
이미 있는 모드를 포함해 모든 모드를 포함시켜주세요. 현재 모드 디렉토리를 삭제하고 이 리스트로 대체합니다.
설정 파일을 저장하고 서버로 업로드하세요.

서버에서 마인크래프트 서버가 설치된 디렉토리로 cd하고, 다음 명령을 실행하세요:
curl -OJ https://raw.githubusercontent.com/jcxmt125/side-projects/main/python%20scripts/kr/mcs-deploy-config/mcs-update-mods.py
설정 파일을 확인 후, 마인크래프트 서버를 정지하세요. (stop을 입력하세요)
스크립트를 실행한 뒤, 설정 파일의 이름을 입력하세요. (서버 셋업 설정 파일도 이해합니다)
잠시 후, 스크립트가 종료되며 모드가 업데이트됩니다. 마인크래프트 서버를 재시작하세요.

클라이언트 측 모드 설치

서버에 모드를 설치했다면, 이제 클라이언트에도 설치할 시간입니다.
그것을 도와줄 스크립트도 준비해 두었습니다!
우선, 클라이언트에 패브릭 로더를 설치하세요.
fabricmc.net에서 더 많은 정보를 찾을 수 있습니다.
그 다음, 클라이언트 측 설정 도우미 스크립트를 다운로드 해주세요.
설정 파일 또한 준비해주세요. 이 스크립트는 서버측 모드 업데이트 스크립트와 동일한 설정 파일 종류를 지원합니다.

스크립트를 파일 내로 이동해주세요. 실행 중 파일을 몇 개 생성합니다.
스크립트를 실행하고, 터미널 창에 설정 파일을 끌어다 놓으세요.
알파벳 이외의 문자나 스페이스가 포함되지 않은 디렉토리인지 확인해 주세요.
스크립트 실행 디렉토리의 "mods.old" 폴더로 현재 모드를 백업합니다.
또한 요청하신 모드를 전부 다운로드합니다.
인터넷 연결에 따라 완료까지 약간의 시간이 걸릴 수 있습니다.
완료되면 메시지를 출력합니다.

마인크래프트 런처를 실행하고, 패브릭 설치를 선택해 실행하세요.
일부 모드는 서버 전용으로, 클라이언트 측에서 효과가 없을 수 있습니다. 일부는 클라이언트 전용으로, 서버 측에서 영향이 없습니다.
모드 페이지의 설명을 참고해주세요.
