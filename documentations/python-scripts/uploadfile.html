파이썬 스크립트

환경 셋업

이 프로젝트는 다음 파이썬 모듈을 필요로 합니다:

  • b2sdk:Backblaze의 서비스를 사용한다면
  • boto3: 다른 S3 호환 프로바이더를 사용한다면
  • python-dotenv
  • requests

이 스크립트는 .env 파일을 필요로 합니다. Backblaze:

  • B2_APPLICATION_KEY_ID
  • B2_APPLICATION_KEY
  • B2_BUCKET_NAME
  • B2_FILE_URL_PREFIX
S3 호환 프로바이더:
  • S3_COMPAT_ENDPOINT_URL
  • S3_COMPAT_ACCESS_KEY_ID
  • S3_COMPAT_SECRET_ACCESS_KEY
  • S3_COMPAT_BUCKET_NAME
  • S3_COMPAT_FILE_URL_PREFIX
Short.io 링크 줄이기:
  • SHORT_IO_API_KEY
  • SHORT_IO_DOMAIN

윈도우 환경을 상정하고 개발되었습니다.
다른 플랫폼의 경우 파일 경로 처리가 바뀌어야 할수도 있습니다.

추가 셋업

Backblaze 또는 다른 S3 호환 프로바이더에 가입해야 합니다.
Short.io에 회원가입하고 도메인을 연결해야 합니다.
S3 호환 프로바이더에서 Lifecycle (자동 삭제) 규칙을 설정해야 합니다.

사용 방법

스크립트를 실행하고 설명을 따릅니다.
