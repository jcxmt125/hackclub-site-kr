Environment configuration

This script requires the following modules:

  • b2sdk:If using Backblaze specific scripts
  • boto3: If using any other S3 compatible providers
  • python-dotenv
  • requests

This script requires a .env file. For Backblaze:

  • B2_APPLICATION_KEY_ID
  • B2_APPLICATION_KEY
  • B2_BUCKET_NAME
  • B2_FILE_URL_PREFIX
For generic S3 provider:
  • S3_COMPAT_ENDPOINT_URL
  • S3_COMPAT_ACCESS_KEY_ID
  • S3_COMPAT_SECRET_ACCESS_KEY
  • S3_COMPAT_BUCKET_NAME
  • S3_COMPAT_FILE_URL_PREFIX
For Short.io link shortening:
  • SHORT_IO_API_KEY
  • SHORT_IO_DOMAIN

This script is designed for Windows environments.
Path-splitting may need to change for other environments.

Further configuration

You need to sign up for Backblaze, or another S3 compatible storage provider.
You need to sign up for short.io and connect a domain.
You need to set up lifecycle rules in your S3 compatible storage provider.
