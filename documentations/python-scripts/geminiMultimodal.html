gemini-api-multimodal.py의 설명서

소스 코드

파이썬 스크립트

환경 셋업

이 스크립트는 추가 파이썬 모듈을 필요로 합니다:

  • google-generativeai
  • python-dotenv
  • pillow

이 스크립트는 .env 파일을 필요로 합니다:

  • GEMINI_API_KEY: Google Gemini API 키

사용 방법

터미널에서: "gemini-api-multimodal.py"를 실행하세요.
이미지 파일을 끌어다 놓고, 엔터를 눌러 첨부하세요.
빈 줄에서 엔터를 눌러 요청을 보내세요.

스크립트를 수정해서 이미지 분석 프롬프트를 변경할 수 있습니다.
아직 이미지만 지원합니다.
