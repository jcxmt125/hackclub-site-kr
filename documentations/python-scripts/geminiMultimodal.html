Environment configuration

This project requires additional modules.

  • google-generativeai
  • python-dotenv
  • pillow

This script requires a .env file:

  • GEMINI_API_KEY: Google Gemini API key

Usage instructions

In a terminal: execute "gemini-api-multimodal.py".
Drag and drop, then return to attach images.
Tap return again to send.

The script can be modified to change the prompt to your liking.
The script only supports images currently.
