소스 코드

파이썬 스크립트

환경 셋업

이 스크립트는 추가 모듈을 필요로 하지 않습니다.

이 스크립트는 .env 파일을 필요로 하지 않습니다.

서버에서 사용

이 스크립트는 리눅스 서버에 마인크래프트 서버의 설치를 도와줍니다.
우선, 파이썬 파일을 마인크래프트 서버를 설치하고자 하는 디렉토리로 이동하세요.
파이썬 파일을 실행합니다. 몇 개의 질문을 물어볼 거에요.
설명을 따라 현재 디렉토리로 마인크래프트 서버를 설치합니다.
완료 후 표시되는 커맨드를 이용해 서버를 시작할 수 있습니다.
완료 후에는 파이썬 스크립트를 삭제하셔도 됩니다.

오라클 무료 티어로 배포하는 법

이 공식 가이드를 기반으로 작성되었습니다.
참고: 오라클을 사용하는 이유는 가장 높은 무료 티어 제한 때문입니다.
이 내용은 언제든 변경될 수 있습니다.
이 가이드를 이용해 발생하는 재정적 문제에 대해 책임질 수 없습니다.

OCS에 가입

무료 티어 설명 웹사이트에 접속해 내용을 조심히 읽어보세요.
무엇이 무료이고 무엇에 비용이 청구되는지 이해하세요.
항상 무료 (Always Free) 카테고리의 "Arm compute instance"를 사용합니단.
무료로 시작하기 (start for free)를 클릭하고, 회원 가입을 시작합니다.

개인 정보를 입력해야 합니다.
동의하는 약관을 자세히 읽어보세요.
모든 경고를 자세히 읽어보세요. "A1 compute instance"와 관련된 경고는 특히 중요합니다.
Home region을 고를 때에는 특히 주의하세요.
경고에 나와있듯이, 나중에 변경할 수 없습니다.
개인 정보를 입력할 때 주의해주세요. 나중에 법적 문제가 생길 수 있습니다.
이전에 말했듯, 이 가이드는 참고용이며, 어떤 보장도 드리지 못합니다.

결제 정보가 필요합니다.
카드에 1.5sgd 내외의 임시 청구가 발생합니다.
비자, 마스터, 아멕스 중 하나의 해외결제가 사용 가능한 카드를 이용해주세요.
무료 체험 시작하기 (start my free trial)를 눌러 시작하세요.
그 후, 계정이 준비되는 동안 잠시 기다리세요.

생성된 계정으로 로그인합니다.
비밀번호 관리자가 비밀번호를 제대로 저장하지 못했을 수도 있으니, 첫 한 번은 직접 찾아서 입력해야 할 수도 있습니다.
2단계 인증 설정을 강력히 권장합니다.

A1 인스턴스 배포하기

계정에 로그인 후, 시작하기 (get started) 탭으로 향합니다. 기본적으로 여기로 리디렉트 될거에요.
"launch resources" 카테고리 아래의 "create a VM instance"를 클릭합니다.
이 가이드에서 말씀드리지 않았다면, 기본값이 정상적으로 설정되어 있을 것입니다.

"image and shape" 메뉴를 찾아주세요.
편집 (edit)을 눌러 편집을 시작합니다.
"change shape"를 누르고, "Ampere"를 선택합니다.
"VM.Standard.A1.Flex" 옆의 체크박스를 선택합니다.
원하는 양의 CPU와 RAM을 배정합니다. 최대 OCPU 4개와 RAM 24GB를 무료로 받을 수 있습니다.
이 옵션은 나중에 수정할 수 있습니다!
"select shape"를 눌러 저장하세요.
"Always Free-eligible"이 계속 표시되는지 확인해 주세요.

다음으로는 SSH를 셋업합니다.
이미 익숙한가요? 키 파일을 업로드해 주세요.
그렇지 않다면, 지금 키 파일을 다운로드 해주세요.
특히 Private key (비공개 키)는 다시 표시되지 않습니다.
이 키 없이는 가상 머신에 접근할 수 없습니다.

저장 공간 설정은 기본으로 유지해도 괜찮습니다.
Create (생성) 버튼을 눌러 가상 머신을 생성하세요.
에러가 발생할 수 있습니다만, 이는 Oracle 측의 용량 부족 문제로, 해결될 때 까지 시간이 필요할 수 있습니다.

마인크래프트 서버를 준비하고 배포하기

서버가 준비되었나요? 이제 SSH로 연결할게요.
공개 IPv4 주소, "Public IPv4 address"가 필요합니다.
"ssh opc@[IP address] -i [다운로드한 .key 파일 경로]" 를 명령줄에 입력하세요.
"yes"를 입력해 이 서버를 신뢰합니다.
연결이 완료되고, 이제 서버에서 명령을 실행할 수 있습니다.

다음으로는 파이썬 스크립트를 다운로드합니다.
우선 원하는 디렉토리로 cd하세요.
다음, 이 명령어를 붙여넣고 실행합니다:
curl -OJ https://raw.githubusercontent.com/jcxmt125/side-projects/main/python%20scripts/mc-serverdeploy.py
다음으로는 Java를 설치합니다.
yum list jdk* 를 실행하고 잠시 기다려주세요.
결과값을 읽은 뒤, 가장 최신 버젼의 JDK를 설치하세요.
jdk-[version-number]-... 와 같이 생겼습니다. 있다면 Headless 옵션을 선택해주세요.
sudo yum install [패키지 이름] 을 실행하세요.
확인창에 y를 입력해 설치를 계속하세요.
그 후, 파이썬 스크립트를 실행해 설명을 따라주세요.

서버를 시작할 준비가 되었습니다.
파이썬 스크립트가 제공한 명령어를 기억하거나 적어두세요.
SSH 세션을 아직 닫지 말고 데스크탑으로 돌아와주세요.

네트워크 설정

이제 서버 연결을 설정합니다.
실행되고 있는 서버의 설정 화면에서, 서브넷 링크를 클릭합니다.
subnet-[date]-[time]과 같은 형식이 기본값입니다.
"Default Security list for..." 을 클릭합니다.
"Add ingress rules" 버튼을 클릭합니다.
Source CIDR 는 0.0.0.0/0, Destination Port range 는 25565로 설정합니다.
"Add ingress rules"를 눌러 저장합니다.

SSH 세션으로 돌아가주세요.
터미널에서 다음 3개의 커맨드를 실행합니다:
sudo firewall-cmd --permanent --zone=public --add-port=25565/tcp
sudo firewall-cmd --permanent --zone=public --add-port=25565/udp
sudo firewall-cmd --reload

선택적으로 더 쉬운 연결을 위해 IP 주소를 도메인에 연결하세요.
Public IP address에 대한 A 레코드를 만들어주세요.
Cloudflare를 사용중이라면 이 레코드에 대해 프록시 기능 (주황 구름)을 꺼주세요.

서버 시작하고 접속하기

스크립트가 제공한 명령어를 실행합니다.
SSH 환경을 접속 해제했을 때 서버가 계속해서 실행되도록 Screen이나 Tmux에서 실행하는 것을 추천합니다.
서버가 곧 시작됩니다!
마인크래프트 클라이언트에서, 서버를 추가해주세요.
멀티플레이를 클릭하고, "서버 추가"를 클릭해주세요.
IP 주소나, 설정된 도메인을 입력합니다.
이제 연결할 수 있습니다!

서버 멈추기

서버를 멈출 필요는 없을 것이지만, 필요하다면, 서버가 실행되고 있는 창에 "stop"을 입력하고 엔터를 누릅니다.
서버의 설정을 변경할 수 있지만, 이 스크립트는 그것까지는 도와드리지 않습니다.
