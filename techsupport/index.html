Not sure what platform you're using?

If you're using a computer, and you aren't using a device from apple, you're likely on Windows.
Try pressing this. If your browser asks whether to allow this site to open settings, you're on windows.
Dismiss the window (press cancel) and head to the appropriate place above.
If it doesn't work and you're on a computer, you may be using an outdated version of Windows.
In this case, this website cannot help, and you should ask someone knowledgable about this topic instead.

If you're using a phone, and it's not an iPhone, you're likely on Android.
Not sure? Turn your phone over to the back. If you see a logo of an apple, you're on an apple device.
