Android help - AOSP / disruptions / pip

Closing the current session

You can simply close the floating pip window.
Drag the window towards the bottom of the screen.
There will be an "X", or another icon indicating that it will close the window.
Drag it within the icon, then release.

Preventing an app from using pip

This will prevent the app from using this floating window.
You'll have to know which app is displaying this.
When you exit an app, and a floating window is spawned with the contents, that app is using the Picture-in-picture feature.
Locate the app, then click in the "i" icon or the "app info" button to open its app info page.
Scroll down to find "Picture-in-picture" under "Advanced".
Tap, and toggle it off.

You can follow a similar procedure to enable pip again at a later date.
Simply toggle the feature back on, following the previous steps.
